# 🌤 Weather Dashboard

A real-time weather application that fetches live weather data using the OpenWeather API. Users can search any city and view live weather conditions.

---

## 🚀 Features
- Search weather by city name
- Real-time API integration (Fetch + async/await)
- Displays temperature, humidity, wind speed
- Error handling for invalid city names

---

## 🌐 Live Demo
https://transcendent-baklava-66e2a2.netlify.app/

---

## 💻 Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
